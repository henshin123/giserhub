这是什么

这是个用来分享自己想法的网站

发起问题

你可以在社区中提出自己的问题，会有人为你解答疑问，让你的问题迎刃而解，但是你的问题需要经过审核其他人才能看到

上传资料

你可以将你的资料发布，让他人能够得到你的资源，让GIS的学习方式更多，让学习渠道来的更简单

分享想法

你可以分享自己的想法，将自己的想法告诉其他人，让他人可以为你的想法增添色彩

组建团队

在发起想法的同时你可以发起团队的创建，规定人数，这样其他人可以联系你跟你组队，让你的想法快速实现

存储想法

你可以建立自己的想法库，并将其设为私有，这样你的想法可以保存在自己的库中，为您以后使用

想法地图

使用app，在发布想法的同时可以在你所在的地方留下的你的想法，当其他人来到这附近时。可以看到你的想法并和你交流

发布问题 & 上传资料
建立自己得想法和团队

通过网站我们可以发布困扰自己的问题，评论他人的问题，为他人的问题进行解答。 并且可以上传你手上拥有的资源，同时你可以下载他人的资源。 发布自己的想法，组建自己的团队，让自己的想法能够实现
